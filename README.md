# ❤️ Heart Disease Risk Prediction using Machine Learning

This project predicts the risk of heart disease using Machine Learning based on medical attributes such as age, cholesterol level, blood pressure, heart rate, etc.

## 📌 Project Overview
The goal of this project is to build a Machine Learning model that can predict whether a person is likely to have heart disease based on health parameters. This project demonstrates the complete ML workflow including data preprocessing, model training, evaluation, and prediction.

## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Scikit-learn
- Logistic Regression
- Google Colab / Jupyter Notebook

## 📊 Dataset Information
The dataset contains medical attributes such as:
- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- Rest ECG
- Maximum Heart Rate
- Exercise Induced Angina
- Oldpeak
- Slope
- Number of Major Vessels
- Thalassemia
- Target (Heart Disease Risk)

## 🤖 Machine Learning Model
Model Used:
- Logistic Regression

## 📈 Model Accuracy
- Training Accuracy: ~85%
- Testing Accuracy: ~80%

## 🔍 Project Workflow
1. Data Collection
2. Data Preprocessing
3. Feature & Target Splitting
4. Train-Test Split
5. Model Training (Logistic Regression)
6. Model Evaluation
7. Prediction System

## ▶️ How to Run the Project
1. Clone the repository
2. Open the notebook in Google Colab or Jupyter Notebook
3. Upload the dataset (heart-disease.csv)
4. Run all cells to train the model and make predictions
